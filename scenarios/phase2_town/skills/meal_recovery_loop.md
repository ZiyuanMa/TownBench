# Meal Recovery Loop

Use this loop when cash is low:

1. Buy one `meal_ingredients` batch for 2 at the `ingredient_seller`
2. Convert it into one `meal_box` at the `meal_prep_table`
3. Deliver the `meal_box` to the `meal_counter` for 4

This is the safest fallback loop because the required capital is low and the item count stays small.
